Basic Steps GIt :

1)Create Remote Repo.
2)Create Directory on local.
3)Initialize Git. "git init".
4)check status of non added files "git status".
5)for red ones add them into stage "git add FileName or . for all".
6)Now finally commit all changes to local repository "git commit -m(branch name) and commit_message".
7)Add remote repository using "git remote add remoterepo_friendlyname https://github.com/user/repo.git".
8)finally push changes to remote repository "git push origin master". & then enter uname & password.
